## Requisitos

Antes de começar, certifique-se de ter as ferramentas abaixo instaladas:
- **Java 17+**
- **Maven 3.6+**
- **MySQL 8+**
- **Dump do banco de dados (na raiz do projeto)**
- **Postman ou cURL (para testar os endpoints)**
